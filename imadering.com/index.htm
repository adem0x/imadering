О программе ...

Основная идея написания этой программы следующая:
Создать клиент для всех популярных протоколов общения в сети интернет. Чтобы этот клиент максимально экономил интернет трафик и ресурсы системы. Чтобы он быстро работал. Чтобы в нем были реализованы различные дополнительные возможности протоколов. Также отсутствие рекламы. Вот основные задачи.
Конечно, есть конкурентные разработки, такие как: QIP, Miranda, R&Q. Но у них нет всех вышеперечисленных качеств вкупе. Постоянно чего-то не хватает, а реализовать в них это не возможно по разным причинам. Например, из-за закрытости исходного кода (как с QIP). Поэтому было решено писать всё самому, чтобы иметь полную власть над программой и уже самому убедиться в том, что можно реализовать, а что нет.

Реализованные возможности программы:

Поддержка операционных систем семейства Windows, начиная c Windows 2000.

Открытость исходного кода программы.
  • Программа написана на самой новой версии языка программирования Delphi 2010.
  • В программе используются бесплатные компоненты с открытым исходным кодом.
  • Исходный код периодически обновляется в SVN репозитории проекта.
  • Любой желающий может внести свой вклад в развитие программы.


Программа работает быстро и потребляет мало системных ресурсов.
  • Программа запускается за 1 - 2 секунды.
  • Автоматическая очистка оперативной памяти занимаемой программой.
  • Минимальное потребление GDI объектов.
  • Возможность запускать программу с высоким приоритетом.


Поддержка различных языков интерфейса программы.
  • Встроенный автоматический переводчик сообщений в чате на 52 языка мира через сервис Google Translate API.
  • Смена языка интерфейса программы без перезапуска.


Поддержка ICQ протокола последней (расширенной) редакции.
  • Просмотр и установка самой полной анкетной информации в сети ICQ.
  • Возможность использовать дополнительные функции протокола, которые отсутствуют в официальном клиенте.
  • Отображение до 200 временных контактов из серверного списка учётной записи ICQ (помогает восстановить случайно утраченные контакты).
  • Возможность отправки произвольных пакетов протокола (для исследователей).
  • Возможность удалить себя из чужого списка контактов.


Поддержка Jabber протокола.

Поддержка Twitter протокола.

Поддержка "Mail.ru агент" протокола.

Окно чата с закладками и поддержкой анимированных смайликов.
  • Быстрое воспроизведение анимации смайликов.
  • Возможность цитировать выделенный текст сразу в ваш Twitter.


Поиск новых контактов по всему миру.
  • Расширенные варианты поиска контактов в сети ICQ.


Хранение истории сообщений и поиск по ней.

Передача файлов через веб сервисы.

Подсчёт и просмотр потраченного программой интернет трафика.

Поддержка подключения через прокси: HTTP, HTTPS, Socks.

Поддержка смены тем иконок в программе и различных наборов смайликов.

Поддержка различных звуковых схем для событий в программе.

Автоматическое создание архивов истории сообщений сжатых 7zip.

Поддержка профилей при запуске программы.

Полная поддержка юникода в программе.

Повышенная защита программы от проникновения внешнего вредоносного кода.

Оптимизация под Wine для работы в операционных системах Linux.

Встроенный в программу монитор для изучения данных протоколов.

Встроенный в программу отладчик для формирования отчётов об ошибках.

Автоматическая проверка обновлений программы.
  • Оповещение при доступности новой версий программы.
  • Возможность обновлять программу скачивая новую версию с официального сайта.
  • Возможность обновлять программу встроенным механизмом обновления.


Поддержка дополнительных расширений программы "Plugins" (в разработке).

Это не полный список возможностей программы, которые постоянно дополняются.
