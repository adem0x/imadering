--- Demo plugin for IMadering ---

- EN -
This plugin displays the channels and data, which go from program IMadering to plugins.
Also he allows to send given from plugin in program IMadering.

- RU -
Этот плагин отображает каналы и данные, которые транслируются из программы IMadering к плагинам.
Также он позволяет отправлять данные от плагина в программу IMadering.
